Cloud Volumes ONTAP 支持 NetApp 卷加密（ NVE ）和 NetApp 聚合加密（ NAE ）。NVE和NAE是基于软件的解决方案、支持FIPS 140-2合规的卷空闲数据加密。 link:concept-security.html["详细了解这些加密解决方案"]。

外部密钥管理器支持 NVE 和 NAE 。
