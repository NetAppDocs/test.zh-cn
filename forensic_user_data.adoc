= 取证用户数据
:hardbreaks:
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media


[role="lead"]
" 用户 " 页面用于标识 Cloud Secure 监控的用户。

此页面中提供的数据包括：

* 由 Cloud Secure 监控的每个用户，并具有指向的链接 link:user_profile.html["用户配置文件"]。
* 由用户执行的活动引发的警报。警报严重性标识如下： image ：： critical.png[ 严重 ] + \ image ：： warning.png[ 警告 ] + \ image ：： low.png[ 低 ] + \
* 用户上次执行的活动。
* 用户的位置—国家 / 地区和省 / 自治区 / 直辖市。
* 用户所属的社区数量。
