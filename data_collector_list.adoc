= Cloud Insights 数据收集器列表


[role="lead"]
Cloud Insights 支持许多供应商和服务提供的各种数据收集器。

数据收集器按以下类型进行分类：

* 基础架构：从存储阵列，交换机，虚拟机管理程序或备份设备等供应商设备中获取。
* 服务：从 Kubernetes 或 Docker 等服务中获取。也称为 _Integration_ 。
