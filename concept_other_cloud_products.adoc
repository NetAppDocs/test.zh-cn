= 正在切换到其他 Cloud Central 产品
:allow-uri-read: 


[role="lead"]
您可以通过单击 * 云中心 * 菜单并从列出的 NetApp 云产品中进行选择来切换到其他 NetApp 云产品。您也可以选择转到 Cloud Central 门户本身。

image:CloudProductsMenu.png["其他云产品"]

您将自动通过身份验证并登录到选定的云产品。
